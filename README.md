# Made in My Deesa
An e-commerce platform for local businesses in Deesa.
## Features
- Google Sheets integration for product catalog.
- - Razorpay payment gateway integration.
  - - Cart and checkout system.
    - ## Setup
    - 1. Update js/app.js with your Google Sheet CSV URL.
      2. 2. Update the appsScriptUrl if you want to receive email notifications.
         3. 3. Deploy on GitHub Pages.
